Rest
Modelo de arquitetura evolução do Http
* Principais Methodos(possui mais)
Get - obeter recurso HTML
Post - criar um recurso
Put - atualizar
Delete - remover
Trace e connect 

Rest determina boas praticas com tabelas de respostas(estados)
1xx - Informativo - Continue
2xx - Sucesso - OK
3xx - Redirecionamento - Moved Permanently
4xx - Erro cliente - Bad request
5xx - Erro de Servidor - Internal Serv error
